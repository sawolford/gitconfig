NAME

git finishfrom

SYNOPSIS

git finishfrom branch - rebase from first argument, fast forward merge, then delete first argument

DESCRIPTION

git finishfrom is a helper alias to rebase another branch onto HEAD and then delete that branch.

  1. Ensures that the current HEAD is not a detached HEAD
  2. Ensures that the proper number of arguments have been provided
  3. Ensures that the provided argument is associated with a branch
  4. Calls the rebasefrom alias to rebase the changesets to the source branch
  5. Calls the mergefrom alias to fast-forward merge the HEAD with the source branch
  6. Deletes the branch since it is not no longer needed

EXAMPLES

  • git finishfrom branch

SEE ALSO

finishto rebasefrom mergefrom

DETAILS

finishfrom = "!f() { : git man finishfrom; \
    if ! git symbolic-ref HEAD; then echo 'on detached head'; return 1; fi; \
    [ $# -ne 1 ] && echo 'Usage: finishfrom branch' && return 1; \
    if ! git show-ref $1; then echo 'branch not a named ref'; return 1; fi; \
    git rebasefrom $1; \
    git mergefrom $1; \
    git branch -d $1; \
}; f"
