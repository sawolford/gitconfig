git alias.gitconfig(1)

NAME

gitconfig

SYNOPSIS

gitconfig - gitconfig aliases

DESCRIPTION

  • git backup - create $(basename $(pwd)).X.tar, where X increases with each invocation
  • git backuprm - remove all backups from previous backup invocations
  • git backups - list available backups from previous backup invocations
  • git branchpullrebase branch - stow working copy, check out first argument, then execute pullrebase
  • git charsets - print number of tracked files of each likely charset
  • git filetype - execute unix file command on all tracked files
  • git finishfrom branch - rebase from first argument, fast forward merge, then delete first argument
  • git finishto branch - rebase to first argument, fast forward merge, then delete first argument
  • git forceeol - force a file pattern to have a given eol setting
  • git gitbinary - print tracked files that git considers binary
  • git gittext - print tracked files that git considers text
  • git globalhookspath - add core.hooksPath to ~/gitconfig/local
  • git man command - output the man page for a git command in the best format possible
  • git masterpullrebase branch - execute branchpullrebase on master
  • git mergefrom branch - stow working copy, merge with first argument, then unstow
  • git mergeto branch - stow working copy, merge with first argument, then unstow
  • git mimetype - execute unix file –mime command on all tracked files
  • git nonascii - print non-ascii lines in a grep-like way
  • git nonasciiall - print non-ascii lines for all tracked files
  • git notasciilist - list all tracked files with non-ascii characters
  • git pickfrom branch - stow working copy, cherrypick changesets, then unstow
  • git pullrebase - fetch, then rebase HEAD with its upstream branch
  • git rebasefrom branch - stow working copy, rebase first argument onto HEAD, then unstow
  • git rebaseto branch - stow working copy, rebase HEAD onto first argument, then unstow
  • git rewindbase source - move a branch to the common ancestor with its upstream
  • git smf command - iterate over submodules to execute shell command
  • git smfa command - iterate over submodules and current directory to execute shell command
  • git smg command - iterate over submodules to execute git command
  • git smga command - iterate over submodules and current directory to execute git command
  • git squash - squash changesets and recommit
  • git stow - save the workspace state
  • git stowco branch - stow working copy, checkout first argument, then unstow
  • git stowcob branch - stow working copy, checkout first argument that tracks second argument, then unstow
  • git stowreco - stow working copy, re-checkout HEAD, then unstow
  • git uncommit - uncommit changesets and restore to working copy
  • git unstow - restore the workspace state
